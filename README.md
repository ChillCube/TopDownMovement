# TopDownMovement

A godot addon used to create top down movement. Can be used for both player characters and NPCs

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/TopDownMovement.git
```

## Documentation
https://github.com/ChillCube/TopDownMovement/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon TopDownMovement
```
